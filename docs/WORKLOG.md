# Work Log

이 문서는 프로젝트를 진행하면서 어떤 판단을 했고, 무엇을 만들었고, 다음에 무엇을 할지 기록하는 작업 로그입니다.

포트폴리오에서는 결과물만큼 과정이 중요하므로, 큰 작업이 끝날 때마다 이 문서를 업데이트합니다.

## Log Rule

각 기록은 아래 형식을 기준으로 남깁니다.

```md
## YYYY-MM-DD - 작업 제목

### 목표
- 이번 작업에서 해결하려는 문제

### 변경 사항
- 실제로 추가/수정한 내용

### 배운 점 / 이슈
- 시행착오, 판단 근거, 기술적 메모

### 다음 단계
- 이어서 할 작업
```
